filesystem
==========

## Usage

Define `tomk79/filesystem` in your `composer.json`.

```
{
    "require": {
        "php": ">=5.3.0",
        "tomk79/filesystem": "dev-master"
    }
}
```

Execute `composer install` command.

```
$ composer install
```

Or update command.

```
$ composer update
```



## Test

```
$ cd (project directory)
$ ./vendor/phpunit/phpunit/phpunit php/tests/filesystemTest
```
